# My Library

A sample app using Java `CUBA-Plateform` and `MySQL`

### Pre-reqs

- JDK 1.8
- IntelliJ / CUBA-Studio

#### Functionality

- A `RESTful CRUD` mock api with response format `JSON`
  
  

#### Dependencies

- CUBA Studio v6.10.0
- [MySQL Script](https://www.dropbox.com/s/vtp6yesnb1k2zst/my_lib_0.sql?dl=0)
- Gradle
